# Portfolio

Personal portfolio website built with `React + TypeScript + Vite`.

## Stack

- `React`
- `TypeScript`
- `Vite`
- `ESLint`

## Quick Start

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` - start development server
- `npm run build` - create production build
- `npm run preview` - preview production build locally
- `npm run lint` - run linter

## About

This repository is the foundation for a personal portfolio project focused on clean architecture, readable code, and solid development practices.
